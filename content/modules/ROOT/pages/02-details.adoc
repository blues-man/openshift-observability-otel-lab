== Application Namespace

The dice game application runs in the `tutorial-application` namespace. All lab exercises deploy workloads into this namespace.

== Terminal Access

The lab terminal is embedded directly in the Showroom interface. Click the *Terminal* tab in the panel to open a pre-authenticated shell connected to the lab cluster.
